화면 배치 방법 : display

        요소가 화면에 어떻게 보여질지를 지정하는 속성 
        
        - block : 화면을 수직 분할(행으로 나눔)
        - inline : 화면을 수평 지평순활속성(열로 나눔)
        - inline-block : inline의 수평분할
        - none : 화면에 요소가 표시되지 않게 하는 속성
        - flex : 요소의 정렬되는 방향이나 요소의 간격을 유연(flex)를 하게 되어주려는
