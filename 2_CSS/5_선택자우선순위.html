CSS 선택자의 우선 순위

        기본적으로 css속성은 style 태그 Ehsms csss 파일에 순서(위->)대로
        해석이 진행되지만, 같은 요소에 따라 css 속성이 설정되는 경우 우선순위가 적용된다.

        * 알아둬야 할 것
        1) 동일한 우선 순위로 css속성이 설정되는 경우, 제일 마지막에 css속성이 반영된다.
        2) 여러 선택자를 이용해서 같은 요소를 선택하더라도, 지정되는 css속성이 다르면 모두 반형된다.


    1순위 : css속성 : 속서앖 limportant
    2순위: inline-style 속성(요소에 직접 작성하느 ㄴpx)
    3순위 :: 아이디 선택자(#아이디명)
    4순위 : 클래스 선택자
    5순위 : 태그선택자
